# Observer attempts to create a work item.
# BEHAVIOR: rd create succeeds locally but campfire send is blocked.
# The item is buffered to pending.jsonl — not propagated to other members.
# The warning 'campfire send failed (buffered to pending.jsonl): not a member'
# is the enforcement signal. Write isolation is at the campfire layer.

$ CF_HOME=$OBS_CF  rd create 'Sneaky item' --type task --priority p3  (blocked at campfire layer)
rdtestobserverprojd8f-00b

# NOTE: The item was written to the project's local mutations.jsonl (local-first
# architecture). Because the observer shares the project directory, the item
# is visible to anyone reading that directory. The campfire send was blocked —
# the item was buffered to pending.jsonl and was never delivered to remote members.
# Write isolation is enforced at the campfire layer, not the local JSONL layer.
